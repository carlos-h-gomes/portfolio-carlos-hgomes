Utility APIs

Coleção de endpoints para automações (OCR, validações, formatação).

Endpoints

GET /ocr/extract → text
POST /email/send → ok
POST /format/slugify → string
POST /validate/cnpj → bool

Exemplo

curl -X POST https://api.example.com/validate/cnpj \
  -H "Content-Type: application/json" \
  -d '{"cnpj":"12.345.678/0001-95"}'
