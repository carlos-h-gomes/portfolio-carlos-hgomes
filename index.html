Soluções aplicadas

OCR para Documentos Fiscais

Desenvolvimento de uma solução que extrai automaticamente dados de comprovantes da Receita Federal (como CNPJ, razão social, data de abertura, etc.) usando OCR com Python e Tesseract. Aplicado para leitura automatizada de documentos enviados por clientes.

Mini API JSONPath para consultas inteligentes

Construção de uma API REST que recebe objetos JSON e expressões JSONPath, retornando resultados sob demanda. Utilizado como ferramenta de apoio interno para buscas dinâmicas em dados complexos.

Organizador automático de fluxogramas

Solução desenvolvida para organizar automaticamente blocos sobrepostos em fluxogramas do diagrams.net (Draw.io), otimizando a visualização de fluxos extensos criados manualmente por analistas.

HUB de Serviços Internos

Plataforma criada para centralizar ferramentas internas e APIs utilitárias em um único ambiente. Permite que colaboradores executem tarefas como formatação de texto, extração de dados e integrações sem depender do time de tecnologia. Reduziu carga de atendimento técnico e acelerou processos operacionais.

Consulta de Tickets Externa

Desenvolvimento de uma solução que permite a consulta de status de tickets abertos por clientes, com base em filtros simples como e-mail ou número de protocolo. Criado para atender demandas de clientes, suprindo uma limitação da plataforma original e melhorando a transparência com o cliente final.
